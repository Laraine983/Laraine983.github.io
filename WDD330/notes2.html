Notes: Ch 2 - Programming Basics

Use of const and let are block-scoped, meaning they only apply to the block they are used in.


Global scope variables = accessible throughout whole program. Use minimally.


Data types = strings, symbols, numbers, Booleans, undefined and null.





Notes: Ch 3 - Arrays, Logic, Loops

Array = ordered list of values, i.e. cats[0] = 'Calico'; cats[1] = 'Tabby' orconst cats = ['Calico', 'Tabby'];


    Array actions:
  • .length (i.e cats.length;) find certain item in array or change length of array
  • .pop (i.e. cats.pop();) removes last item in array
  • .shift (i.e. cats.shift();) removes first item in array
  • .push (i.e. cats.push();) adds new value to end of array
  • .upshift (i.e. cats.upshift();) add new value to beginning of array
  • .reverse (i.e. cats.reverse();) reverses order of array
  • .sort (i.e. cats.sort();) sorts items in array
