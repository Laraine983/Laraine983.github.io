Ch 11: Further Functions

    Function properties and methods

  • call() method - set the value of this> inside a function to an object that is provided as the first argument
  • if function does not use this in body, call() method must have null as first argument i.e. function.call(null, 5)
  • Custom properties
    - description (i.e. function.description) allows you to describe what function does
    - memoization - saves result in cache property to be used again later (i.e. function.cache)

    Immediately Invoked function expressions

  • function invokes as soon as defined. Use parenthesis at end of function i.e.
    (function(){ const cat = 'Tabby'; console.log(`Cute ${cat}`);})();
  • to use strict mode, place all code in IIFE (especially if collaborating with others) and use strict mode within that
