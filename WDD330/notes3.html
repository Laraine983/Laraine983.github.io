Object Methods: this

  • Methods can use the this keyword to access the object.
  • OOP - object-oriented programming (writing code using objects to represent things)
  • this can be used even if not method of object
  • Object literal - object created by wrapping its properties in {}

    Ch 5: Objects

  • EXAMPLE:
    const aboutMe = {
    name: 'Sierra',
    height: 69,
    weight: 180,
    age: 23,
    location: 'USA',
    faveColors: ['teal','pink','yellow']
    };
  • to access each property, I would use aboutMe.property, i.e. aboutMe.name, aboutMe.height, etc...
  • add new property using same notation, i.e. aboutMe.job = 'student'; adds that property to object

    Ch 6: Document Object Model (DOM)

  • represents an HTML document as connected network, comparable to a tree
  • node type can be found by using nodeType (i.e. body.nodeType)
  • each node holds a numerical value:
    element = 1
    attribute = 2
    text = 3
    comment = 8
    body = 9
  • getElementById() grabs HTML element by its id attribute
  • getElementsByTagName() lists all elements with given tag name
  • getElementsByClassName() lists all elements with given class name
  • getAttribute() gives value of provided attribute
  • setAttribute() can change value of attribute
  • navigate DOM with parentNode(), previousSibling(), nextSibling(), childNodes(), and children()

    Ch 7: Events

  • events link web page to user interaction
  • event listeners listen for event to occur and react as programmed
  • onclick links user interaction to javascript
  • remove event using removeEventListener
