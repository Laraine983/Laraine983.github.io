Object Methods: this

  • Methods can use the this keyword to access the object.
  • OOP - object-oriented programming (writing code using objects to represent things)
  • this can be used even if not method of object
  • Object literal - object created by wrapping its properties in {}

    Ch 5: Objects

  • EXAMPLE:
    const aboutMe = {
    name: 'Sierra',
    height: 69,
    weight: 180,
    age: 23,
    location: 'USA',
    faveColors: ['teal','pink','yellow']
    };
  • to access each property, I would use aboutMe.property, i.e. aboutMe.name, aboutMe.height, etc...
  • add new property using same notation, i.e. aboutMe.job = 'student'; adds that property to object
