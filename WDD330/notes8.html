Ch 08: Transforms and Transitions

    Transforms

  • trasnform property lets you move, rotate, scale any element on page

  • transform: translate(x, y); - x moves items from left, y moves items from top
  • for older browsers, add:
    -webkit-transform: translate(x, y);
    -ms-transform: translate(x, y);
  • for simple up/down translations:
    transform: translateX(x);
    transform: translateY(y);

  • scale(x, y); x = horizontal, y = vertical
  • to scale only horizontally/vertically:
    transform: scaleX(x);
    transform: scaleY(y);

  • skew(x, y); function specifies a skew along the x and y axes. Use same as translate & scale

  • transform-origin: changes origin of applied transforms (such as changing them from center)

    Transitions

  • allow the values of CSS properties to change over time, providing simple animations
  • steps to create transition:
    1) Declare the original state of the element in the default style declaration
    2) Declare the final state of your transitioned element; for example, a :hover state
    3) Include the transition functions in your default style declaration using the transition properties, including: transition-property, transition-duration, transition-timing-function, and transition-delay

    Animations

  • control each step with keyframes
  • keyframes create starting or ending point of an animation
  • @keyframes animationName {
    /* put animation keyframes here */
    }
  • animation duration defines length of animation

    Ch. 12: Canvas, SVG, and Drag and Drop

    Canvas

  • allows us to draw through JavaScript
  • in HTML, add canvas tag
  • set width, height, & id/class
  • add styles in CSS with id/class
  • in JavaScript:
    var canvas = document.getElementById("myCanvas");
    var context = canvas.getContext("2d");
    context.strokeStyle = "red";
    context.fillStyle = "rgba(0, 0, 255, 0.5)";
    context.fillRect(10, 10, 100, 100);
    context.strokeRect(10, 10, 100, 100);

    SVG

  • use svg tag
  • viewBox attribute defines the starting location, width, and height of the SVG image
  • desc tag lets you add description
