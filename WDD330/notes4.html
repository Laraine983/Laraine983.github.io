Ch 8: Forms

  • document.forms: returns list of all forms in html document in order of appearance (i.e. let form = document.forms[0];
  • a form with name attribute of 'search' allows user to input search item and submit
  • action attribute of input gives URL to send form to
  • name attribute in input of 'searchInput' allows access to info inside form
  • form.submit() method submit form automatically, or user can submit manually with button type='sumbit'
  • form.reset() method resets form values to what's specified in HTML

    Ch 12: Object Oriented Programming (OOP)

  • Encapsulation - programming logic kept hidden from user, only allowing them to access funtional parts (like a button)
  • Polymorphism - multiple objects can share the same method
  • Inheritance - adding additional features to current object features to improve functionality
  • Prototypes:
    - constructor.prototype
    - Object.getPrototypeOf()
    - __proto__ (not recommended)
    - setPrototypeOf()
    - isPrototypeOf()
    - prototype can be used to declare new methods/properties; used when same properties used for every instance of a class
