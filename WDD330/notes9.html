Ch 09: Window Object

    Browser Object Model (BOM)

  • collection of properties and methods that contain information about the browser and computer screen
  • refer to global variables without using the window object
  • window.alert() method will pause the execution of the program and display a message in a dialog box
  • window.prompt() method will stop the execution of the program and display a dialog that shows a message as well as an input field that allows the user to enter text

    Browser Information

  • window object has a number of properties and methods that provide information about the user’s browser:
    - navigator property - refers to navigator object which contains browser info
    - userAgent property - return info about browser and OS
  • window.location property:
    - contains info about URL
    - href property - returns URL as string
    - protocol property - returns string with info about protocol used
    - host property - returns string describing domain of URL & port number
    - pathname property - returns string of path following the domain
    - search property - returns a string that starts with a “?” followed by the query string parameters
    - hash property - returns a string that starts with a “#” followed by the fragment identifier
    - origin protocol - returns a string that shows the protocol and domain where the current page originated from
  • window.history property can be used to access information about any previously visited pages in the current browser session
  • window.screen object contains information about the screen the browser is displayed on

    Timing Functions

  • window.setTimeout() - first parameter is callback to function, second is number of milliseconds
  • window.clearTimeout() - cancels timeout
  • window.setInterval() - repeats callback function after each given ms interval
